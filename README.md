# lazy_offer
lazy_offer — высокопроизводительный движок на Rust для автоматизации поиска работы. Использует UserBot (MTProto) и LLM для анализа вакансий в Telegram. Авто-отклик при совпадении стека >70%, уведомления в систему и управление через Bot API. 🦀💼


## 🦀 Проект: LazyOffer (lazy-offer)

**Концепция:** Автоматизированный «ленивый» комбайн для поиска работы, который берет на себя рутину мониторинга, оценки и первичного контакта.

### 1. Функциональные возможности
* **Dynamic Auth:** Вход через управляющего бота по QR-коду (генерация PNG в памяти, отправка пользователю).
* **Multi-Account Support:** Управление сессиями нескольких пользователей через `HashMap<UserId, Arc<Client>>` и хранение сессий в файлах `/sessions/{id}.session`.
* **Smart Scoring (LLM):**
    * **> 70% match:** Автоматический отклик с кастомным текстом от ИИ.
    * **> 50% match:** Сообщение в бот с кнопками «Откликнуться» / «Скипнуть».
* **Auto-Responder:** Если рекрутер пишет в личку (ключевые слова «CV», «резюме»), бот уведомляет тебя и предлагает отправить файл одной кнопкой.
* **Desktop Notify:** Нативные уведомления на компьютере через `notify-rust`, если приложение запущено локально.

### 2. Технологический стек
* **Ferogram (MTProto):** Основа для UserBot (чтение каналов, отправка сообщений от твоего имени).
* **Teloxide (Bot API):** Интерфейс управления (прием команд, выдача QR-кода, уведомления).
* **Tokio:** Асинхронная среда для параллельной работы десятков сессий.
* **Reqwest + OpenAI/Anthropic API:** Мозг системы для анализа текста вакансий.
* **Qrcode-generator:** Создание изображений для входа.
